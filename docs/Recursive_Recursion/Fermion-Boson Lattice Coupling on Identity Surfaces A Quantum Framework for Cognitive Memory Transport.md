\[
\hat{H} = \hat{H}_f + \hat{H}_b + \hat{H}_{\text{int}}

- \( \hat{H}_f = -t_f \sum_{\langle i,j \rangle} \psi_f^\dagger(x_i) \psi_f(x_j) + \text{h.c.} \)
